## Installation

To install use PyPi:

```xsh
pip install git+https://github.com/anki-code/jj
```

## Usage

```xsh
echo '{"name": "John Doe", "age": 30, "city": "New York"}' | jj
```
```json
{
    "name": "John Doe",
    "age": 30,
    "city": "New York"
}
```
